Hallo teman-teman coders, pada tutorial kali ini saya akan membahas beberapa kode perintah tmux yang sering dipakai. Nah, bagi teman-teman yang belum install tmux silahkan install tmuxnya dulu, atau klik disini. Silahkan ikuti step-stepnya agar tidak terjadi error. Jika ada error saat penginstallan silahkan komentar dibagian tab diskusi. Oke, bagi yang sudah install tmuxnya, silahkan perhatikan beberapa kode perintah berikut.

  1. tmux
    Untuk memulai menggunakan tmux, teman-teman cukup ketikan kode perintah tmux pada terminal teman-teman. Untuk hasilnya nanti seperti ini:
  2. Ctrl + B Shift + %
    Kode ini digunakan untuk menyeplit atau membagi pane menjadi dua bagian secara vertical. Jadi untuk melakukannya itu teman-teman yang pertama kali tekan tombol Ctrl + B pada keyboard, nah setelah tekan Ctrl + B, teman-teman lanjut menekan tombol Shift + %, jadi dua kali tekan kode perintah. Untuk hasilnya seperti ini:



    Nah, itu kode perintah untuk membagi pane secara vertical. Diingat ya teman-teman :D

  3. Ctrl + B Shift + "
    Untuk kode perintah yang kedua, yaitu fungsinya untuk menyeplit atau membagi pane secara horizontal, kalau kode yang tadi untuk membagi pane secara vertical. Nah, cara melakukannya juga sama teman-teman seperti pada nomer satu, tapi pada kode perintah yang kedua ini setelah Ctrl + B lalu tekan Shift + ". Untuk hasilnya seperti ini:


  4. Ctrl + B O
    Nah, sekarang bagaimana caranya untuk pindah dari pane 1 ke pane 2? Tenang saja, mudah kok teman-teman. Jadi kalau teman-teman mau pindah dari pane 1 ke pane 2, cukup tekan Ctrl + B, lalu tekan huruf O, maka langsung pindah ke pane sebelahnya. Begitupun jika ingin kembali ke pane asal. Teman-teman juga bisa menggunakan kode perintah berikut untuk pindah pane.
    Ctrl + B Arrow Left
    Ctrl + B Arrow Right
    Ctrl + B Arrow Top
    Ctrl + B Arrow Down

    Silahkan dicoba satu persatu ya teman-teman. Nanti tahu perbedaannya sendiri :D
  5. Ctrl + D Sekarang bagaimana caranya untuk keluar dari tmux? Caranya mudah kok teman-teman. Teman-teman cukup tekan Ctrl + D pada keyboard.

Oke teman-teman itulah tadi beberapa kode perintah pada tmux yang akan sering dipakai ketika kita menggunakan tmux. Untuk kode perintah yang lainnya silahkan teman-teman explore sendiri ya. Hehe. Oke, thank you sudah membaca dari awal hingga akhir, semoga bermanfaat buat teman-teman semua. Jika ada kurang paham, silahkan comment aja, atau hubungi saya langsung. Good Luck :D
