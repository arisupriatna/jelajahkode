Apa itu Terminal ?

Command line? pasti sudah tidak asing lagi ditelinga kita. Nah, sekarang apa itu Terminal? jangan berpikir terminal bus ya? hehe. Jadi Command lain dan Terminal itu sama saja. Hanya saja untuk penyebutan terminal sering disebutkan di os unix. Command line atau terminal yaitu text interface untuk sistem operasi komputer. Command line banyak digunakan dikalangan programmer atau developer, karena dengan menggunakan command line semua pekerjaan terselesaikan dengan cepat, khususnya untuk dibidang server. Jika anda masih awam bagaimana cara menggunakan command line, mari kita belajar dari basic terlebih dahulu. Silahkan perhatikan kode-kode perintah command line berikut.

  1. pwd
    pwd digunakan untuk menampilkan nama direktori kerja saat ini. Perhatikan gambar dibawah ini.


    Pada gambar, ketika kita mengetikan kode perintah pwd, maka muncul nama direktori kerja kita saat ini, yaitu berada di /User/arisupriatna.

  2. ls
    ls digunakan untuk menampilkan nama file dan nama folder kita. Perhatikan gambar berikut ini.


  3. cd <nama_folder>
    cd digunakan untuk memindahkan direktori yang anda tentukan. Misalnya seperti berikut:


    Pada gambar di atas, saya melakukan kode perintah cd Documents/ artinya saya masuk ke direktori dengan nama Documents. Untuk kembali ke folder sebelumnya cukup ketikkan kode perintah ($ cd .. ).

  4. mkdir <nama_folder>
    mkdir berfungsi untuk membuat folder baru. Jadi dengan menggunakannya kode ini di terminal, membuat folder baru lebih simple. Contohnya perhatikan gambar berikut:
