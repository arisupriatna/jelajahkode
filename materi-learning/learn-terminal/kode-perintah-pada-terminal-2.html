Kode Perintah Pada Terminal : Part 2

By Ari Supriatna

Posted on April 5, 2018
Hallo Coders! Sebelum belajar jangan lopa berdoa dulu ya!
  • Materi
  • Diskusi


Assalamualaikum.wr.wb
Selamat siang guys, pada kesempatan kali ini saya akan membahas kode perintah pada terminal diantaranya copy, move, rename dan delete. Oh iya, sebelum lanjut ke materi, gimana sudah paham dengan materi sebelumnya? mudah-mudahan sudah paham ya. Kalau sudah paham silahkan baca materi hari ini, jika belum silahkan buka materi sebelumnya lagi, biar benar-benar paham.
Oke guys, mari kita mulai belajar.

  1. copy
    Untuk melakukan copy atau paste file atau folder, biasanya kita cukup klik kanan lalu klik copy atau paste, atau biasanya dengan shortcut CTRL+C dan CTRL+V pada keyboard. Tapi, kali ini kita akan melakukan copy paste pada terminal. Tentunya berbeda dengan yang disebutkan di atas. Untuk melakukan copy paste diterminal cukup mudah. Silahkan perhatikan gambar berikut.



    Pada gambar diatas, bisa dilihat cara copy paste pada terminal. cukup ketikkan perintah cp <nama_file> <folder_untuk_paste>. Diatas, saya melakukan copy file dengan nama index.html dari folder latihan-terminal, dan file tersebut akan saya pastekan di folder Document, dimana folder Document tersebut terletak di /User/arisupriatna/Document. Gimana? mudah bukan?

  2. mv : pindah file atau folder
    Move bisa digunakan untuk memindahkan file, atau biasanya kita menggunakan perintah cut atau CTRL+X pada keyboard. Fungsinya sama saja, memindahkan file atau folder. Tapi move ini bisa digunakan untuk rename juga. Jadi move ini selain bisa untuk memindahkan file, bisa juga untuk mengganti nama file atau folder. Tetapi untuk kode perintahnya sedikit berbeda. Silahkan teman-teman perhatikan pada gambar berikut untuk melakukan pemindahan file.



    Pada gambar diatas, saya melakukan pemindahan file atau move file dengan nama file index.html dari folder latihan-terminal, yang akan di dipindahkan ke /User/arisupriatna. Untuk kode perintahnya hampir sama dengan kode perintah copy, hanya saja kode cp diganti dengan mv.

  3. mv : rename file atau folder
    Kode mv kali ini digunakan untuk merename (ganti nama) file atau folder. Biasanya kita melakukan rename dengan cara klik kanan folder atau file, lalu klik rename, atau bisa juga dengan klik dua kali pada nama file atau foldernya. Tapi kali ini kita akan merename file atau folder melalui terminal. Silahkan teman-teman perhatikan pada gambar berikut untuk melakukan rename file melalui terminal.



    Pada gambar diatas, saya melakukan rename file dengan kode perintah mv <nama_file_lama> <nama_file_baru>. Diatas, saya melakukan rename file index.html menjadi index2.html. Jadi index.html adalah nama file yang lama, dan index2.html adalah nama file yang baru.

  4. rm : delete file
    rm atau remove adalah kode perintah untuk menghapus file atau folder pada terminal. Biasanya kita menghapus file atau folder melalui tombol delete pada keyboard, atau klik kanan pada file lalu klik delete. Sekarang kita akan belajar menghapus file atau folder melalui terminal. Silahkan teman-teman perhatikan pada gambar berikut.



    Pada gambar diatas, saya menghapus file dengan kode perintah rm <nama_file>. Bisa teman-teman lihat pada gambar diatas, saya menghapus file dengan nama file index.html pada folder Documents. Oh iya, untuk menghapus folder, kode perintahnya sedikit berbeda dengan kode perintah untuk menghapus file. Silahkan teman-teman baca pada point selanjutnya.

  5. rm : delete folder
    Pada point diatas kita telah membahas bagaimana cara menghapus file melalui terminal. Pada point kali ini kita akan membahas bagaiaman cara menghapus folder melalui terminal. Karena untuk menghapus folder, kode perintahnya sedikit berbeda dengan kode perintah untuk menghapus file. Silahkan teman-teman perhatikan pada gambar dibawah ini.



    Pada gambar diatas, saya menghapus folder dengan kode perintah rm -d <nama_folder>. Pada kode diatas ada sedikit tambahan kode, yaitu -d, yang artinya directory. Pada gambar, saya melakukan hapus folder dengan nama latihan pada folder latihan-terminal.

Oke guys, itulah beberapa kode perintah pada terminal yang akan sering digunakan. Untuk materi kali ini, cukup sampai disini saja. Apabila ada yang kurang paham,, silahkan komen di tab diskusi atau tanyakan pada grup WA. Terimakasih, semoga bermanfaat. Sampai jumpa di materi selanjutnya ya guys. Good Luck. Happy Coding. :)
