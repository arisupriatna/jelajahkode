Setup Git

Assalamualaikum.wr.wb
Selamat sore teman-teman coders, apa kabar? Semoga sehat selalu ya. Pada kesempatan kali ini kita akan belajar bagaimana cara setup git. Nah, setelah teman-teman sudah memiliki Git di sistem laptop/PC teman-teman, kita perlu melakukan beberapa hal untuk menyesuaikan lingkungan Git teman-teman. Jika di laptop/PC teman-teman belum menginstall Git nya, silahkan teman-teman download dan install terlebih dahulu, atau teman-teman bisa klik disini untuk cara penginstallannya. Oh iya teman-teman, kalian melakukan setup git ini hanya dilakukan sekali saja pada laptop/PC teman-teman, karena setup tersebut akan bertahan diantara peningkat versi. Teman-teman juga bisa mengubahnya dengan menjalankan perintahnya lagi. Oke, berikut kode perintah untuk setup git di laptop/PC.
Hal pertama yang harus teman-teman lakukan adalah mengatur nama pengguna dan alamat email kalian. Ini sangat penting, karena setiap Git berkomitmen menggunakan ini, dan ini akan dipasang permanen kedalam commit yang kalian mulai buat. 

                  
$ git config --global user.name "user name github kalian"
$ git config --global user.email "email github kalian"

Silahkan teman-teman isi dengan user dan email yang benar dan sesuai dengan akun github kalian.
Jika teman-teman ingin memeriksa konfigurasinya, teman-teman bisa menggunakan kode perintah berikut : 
                  
$ git config --list
Dengan menjalankan kode perintah tersebut, maka teman-teman bisa melihat pengaturan yang dapat ditemukan oleh Git.
Contoh: 
                  
$ git config --list
user.name = Ari Supriatna
user.email= arisupriatna@jelajahkode.com
color.status = auto
color.branch = auto
color.interactive = auto
color.diff = auto
...

Oke teman-teman, untuk pertemuan kali ini sampai disini, mohon maaf apabila ada kesalahan. Apabila teman-teman ada yang kurang paham, silahkan tanyakan di tab diskusi atau melalui grup wa. Terimakasih. Good Luck guys. :)
