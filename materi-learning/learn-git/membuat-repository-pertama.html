Membuat Repository Pertama

Assalamualaikum.wr.wb
Selamat siang teman-teman. Pada tutorial kali ini saya akan membahas bagaimana cara membuat repository di git. Mungkin sebagian teman-teman ada yang sudah membuatnya atau juga ada yang belum. Bagi yang belum, tutorial kali ini tepat sekali untuk teman-teman yang baru belajar. Karena disini akan dijelaskan secara step by step. Jadi teman-teman tidak akan kebingungan. Oh iya, teman-teman tahu tidak apa itu repository? Jadi, repository itu wadah atau tempat kalian menyimpan projectnya, kalau di laptop itu istilahnya sebagai foldernya. Jadi didalam folder itu yang akan menjadi tempat file-file project teman-teman. Karena di github tidak bisa langsung menyimpan begitu saja. Harus ada wadahnya, yaitu tadi repository. Oke, langsung aja teman-teman, kali ini kita akan membuat repository baru di git.

  1. Git init
    Silahkan teman-teman buat terlebih dahulu dahulu folder di local, atau bisa menggunakan terminal dengan cara mkdir <namaFolder>. Setelah teman-teman membuat folder di local, lalu masuk kedalam foldernya melalu terminal, dengan cara cd <namaFolder>, oh iya bagi teman-teman yang belum tahu kode perintah di terminal, silahkan pelajari dulu kode perintahnya atau teman-teman juga bisa baca tutorialnya disini. Nah, setelah teman-teman sudah berada di folder yang teman-teman buat, sekarang lakukan inisialisai, dengan kode perintah berikut:

    Apabila teman-teman sudah melakukan inisialisai, maka akan tampil seperti pada gambar.

  2. Create Repository
    Setelah teman-teman membuat folder di local, sekarang teman-teman buat repository di github. Saran saya buat namanya sama seperti di folder local, agar teman-teman tidak bingung.

    Pada gambar saya membuat repositorynya sama seperti pada folder local saya. Untuk description itu optional bisa di isi atau tidak, sesuai kebutuhan teman-teman. Lalu pilih bagian Public, karena kalau private itu harus bayar perbulan, jika teman-teman mau repositorynya private silahkan teman-teman pilih bagian private. Yang terakhir klik button Create repository.
    Jika berhasil, maka akan tampil seperti berikut:

  3. Git remote Sekarang lakukan git remote pada folder teman-teman, dengan kode perintah pada gambar diatas bagian kolom ke 2.

    Setiap membuat repository baru, kita harus melakukan git remote.

  4. Add files
    Selanjutnya, teman-teman tambahkan file baru di dalam folder yang sudah teman-teman buat. Misalnya kita menambahkan file index.html.

    Untuk isinya, silahkan isi dengan html, bebas. Atau teman-teman bisa copy paste kode berikut: 
                          
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
</head>
<body>
  <h1> Ini adalah file project pertamaku </h1>
</body>
</html>

  5. Git add
    Setelah teman-teman menambahkan file baru, sekarang kita add file tersebut, dengan kode berikut.

    Git add ini digunakan untuk menambahkan file yang akan kita simpan ke repository.

  6. Git status
    Untuk melihat apakah file kita sudah di add atau belum, teman-teman bisa menggunakan kode perintah berikut.

    Nah, jika file kita sudah ter add maka akan tampil seperti pada gambar, warna textnya menjadi hijau. Jika belum melakukan add, maka akan tampil seperti berikut.

    Bisa teman-teman lihat, jika filenya belum kita add, akan pada textnya berwarna merah, artinya belum kita add. Jika sudah maka akan tampil seperti pada gambar di atas.

  7. Git commit
    Selanjutnya kita commit filenya. Apa sih commit itu? jadi commit itu istilahnya memberikan pesan pada file project kita. Misalkan teman-teman membuat Aplikasi, nah pasti akan ada perubahan dong,misal menambahkan fitur something, atau apapun itu, melakukan modifikasi, dan lain-lain. Jadi fungsi commit itu memberitahu kepada kita kalau kita melakukan modifikasi, misalnya dengan pesan "update aplikasi" atau "add new fiture" dan lain-lain, yang jelas memberi pesan untuk mengingatkan kita, bahwa kita telah memodifikasinya. Nah, bagaimana cara melakukan hal tersebut, baik berikut kode perintahnya untuk melakukan commit.

    Pada gambar, saya melakukan commit dengan pesan "First Project", hal itu menandakan itu adalah project pertama saya.

  8. Git push Nah, ini adalah langkah terakhir, yaitu git push. Git push ini yaitu proses upload ke github. Untuk kode perintahnya sebagai berikut.

    Dan akhirnya, repository berhasil dibuat. Selamat :D

Gimana teman-teman caranya mudah bukan? Nah, sekarang teman-teman berhasil membuat repository baru di github. Terimakasih sudah mengikuti tutorial ini dari awal hingga akhir, semoga bermanfaat. Jika ada yang kebingungan atau apapun itu, silahkan comment di tab diskusi atau hubungi saya langsung. Good Luck. :)P
